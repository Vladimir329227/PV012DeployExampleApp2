# PV012DeployExampleApp2
<p>Тестовый проект запуска приложения в облачной среде Amvera </p>
Ссылка на работающий пример : https://pv012deployexampleapptestdocker-vladimir961.amvera.io/
<p></p>
<p>Инструкция по запуску приложения:</p>
<ol>
  <li>Клонировать репозиторий
  (<code>git clone https://github.com/Vladimir329227/PV012DeployExampleApp2</code>)
  </li>
  <li>Создать свой репозиторий с этим проектом</li>
  <li>Загрузить проект на amvera
    <p><code>git remote add amvera https://git.amvera.ru/<Имя учетной записи>/<название проекта на amvera></code></p>
    <p><code>git push amvera master</code></p>
  </li>
</ol>
При вознекновении ошибок: https://docs.amvera.ru/books/amvera/page/podklyucenie-k-sushhestvuyushhemu-repozitoriyu
